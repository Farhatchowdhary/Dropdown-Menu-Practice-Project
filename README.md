# Dropdown Navigation Practice Project

A simple HTML, CSS, and JavaScript project to practice building a multi-level dropdown navigation menu with emoji-enhanced menu items and styled arrows.

---

## Features

- Multi-level dropdown menu
- Submenus open to the right of parent menu items
- Custom colored arrows (`>`) for parent items
- Emoji icons for menu and submenu items
- Fully responsive and easy to customize
- Hover effects on submenu items
- Toggle visibility with a button for the main menu
